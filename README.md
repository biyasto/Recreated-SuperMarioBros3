## CONTROLS

S : Enter Game


UP-DOWN-LEFT-RIGHt: move

A - HOLD: RUN

A - TAP: Attack

S,X : JUMP

S -HOLD: High Jump

S -TAP: Fly/Float


1-4: Change Mario Power

5-9: Move to locations

ESC: Pause/Unpause
